## üìã What is interviewpilot-ai?

interviewpilot-ai is a tool that helps you practice for job interviews using artificial intelligence. It lets you take real-time voice interviews that change based on your answers. After each interview, you get a transcript and a report showing how you did. This makes it easier to prepare for real interviews by understanding your strengths and areas to improve.

This application works on modern computers and runs through your web browser, so you do not need to install anything extra beyond a web browser.
